1) Qu'est ce que le langage Html?

Html signifie "HyperText Markup Language" qu'on peut traduire par "langage de balises pour l'hypertexte".
Il est utilisé afin de créer et de représenter le contenu d'une page web et sa structure.

2) quel est l'histoire de ce langage ?

Le langage HTML a été inventé par Tim Berners-Lee dans les annéés 1990 (1993).
Durant cette période, HTML s'est agrandi de nombreuses manières différentes.

Html n'est pas un langage de progammation (comme le Python par exemple),pas question de conditions ou de boucles... C'est un langage de description.

5) Quels sont les marqueurs du langage HTML ?

Les marqueurs essentiels HTML au début d'un document sont "HTML" ,HEAD" et "BODY" ,il a aussi "mark" qui peut souligné.

6) Les principales balises et leurs fonctions ?

Les principales balises sont "HTML","HEAD","TITLE","BODY" ...
"HTML" début et fin du fichier,"HEAD"zone d'en-tête d'un fichier,"TITLE" titre affiché par le navigateur et "BODY" début et fin du corps d'un fichier.

7) Aujourd'hui l'usage de quelle version de HTML est recommandée par le W3C ?

Le World Wide Web Consortium (W3C) vient de finalement recommander le HTML5. Concrètement, cela signifie que cette nouvelle version de ce langage est suffisamment aboutie pour être démocratisée.
Après avoir réuni plus de 60 compagnies autour de ce projet et avoir résolu 4 000 bugs.

8) En quoi le HTML permet le web sémantique ?

L'objectif du HTML sémantique est de révéler la signification du contenu d'une page Web à la fois au navigateur Web et au développeur, plutôt que de simplement définir l'apparence de la page.

9) Les langages MarkDown et Latex ?

Le Markdown va vous permettre d'écrire en HTML sans pour autant en maîtriser le langage ! Il est très utile dans les milieux de la communication, de la publicité, ou du journalisme. Il permet une mise en forme rapide et pratique.
LATEX est un langage qui permet de créer des documents tout en séparant la forme du fond.Latex est un langage de programmation, qui génère plusieurs types de fichiers.
